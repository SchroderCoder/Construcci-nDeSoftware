Preguntas Lab 6 class

Pregunta 1: ¿Por qué es una buena práctica usar JavaScript para checar que sean válidos los inputs de las formas antes de enviar los datos al servidor?

Por la experiencia del usuario. De esta manera evitamos que se necesite confirmar los datos con conexión de internet y podemos aceptar o declinar el formato inmediatamente.

Pregunta 2: ¿Cómo puedes saltarte la seguridad de validaciones hechas con JavaScript?

Hay 2 simples maneras: podemos desactivar javascript en nuestro navegador y podemos entrar al html y borrar la lectura del script.

Pregunta 3: Si te puedes saltar la seguridad de las validaciones de JavaScript, entonces ¿por qué la primera pregunta dice que es una buena práctica?

Para la experiencia del usuario común, es necesario seguir validando sin ocupar una conexión para reducir el tiempo de espera para aquellos que quieran usar nuestra aplicación como fue diseñada.
