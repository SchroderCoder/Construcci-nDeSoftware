Preguntas Lab 5

  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    • Las similitudes terminan en el nombre, aunque en sus primeras iteraciones, java y javascript tenían una sintaxis simlar. En las diferencias, javascript es un lenguaje interpretado, mientras que Java es interpretado y compilado. También Java es tipado mientras que Javascript no lo es. Java esta enfocado en la OOP y javascript en prototipado y paginas web.

  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

    • El objeto date permite utilizar el tiempo para la medición de ejecución y otras funciones, entre ellas: getFullYear(): devuelve el año en 4 digitos. Date.now(): devuelve la hora en el instante. getTime(): devuele el número de milisegundos desde enero 1, 1970. getDate(): devuelve el día como 2 digitos. getHours: devuele la hora.

  • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

    • Los arreglos nos permiten guardar datos en forma de lista. Entre sus principales métodos están: push(): permite agregar un elemnto al array hasta el final. unshift(): insertar elemento al principio del array. pop(): borrar un elemento del final del array. shift(): remover un elemento del incio del array. Y length(): devuelve el tamaño del arreglo.

  • ¿Cómo se declara una variable con alcance local dentro de una función?

    • Debemos utilizar la sentencia var mas el nombre de nuestra variable.

  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    • Es posible que el valor de la variable afecte la ejecución de otras funciones con variables comunes como "count" o "i" ya que esta guardaría su último valor. Es mejor utilizar variables por referencias
